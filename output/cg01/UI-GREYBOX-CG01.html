画廊 A — 屏（1080×1920 视口帧）

分区：TopBar/History y0–90 / GameView y90–1056(演出区) / PlayerHUD y1056–1920(h864)。GameView 与上方条同处 ContentView(0–1156)，PlayerHUD 与之 重叠 100px(1056–1156)。余额在底部 HUD(MoneyContainer)，顶栏只有历史条 + 齿轮。单注栏：仅 BetSwitcher(±) + 一个 PlayButton + 一个 AutoCashout（无 A/B 双注、无 quick-bet 行）。飞行 3 段(lv1/2/3)= bg 缩放 + 火球/速度线/模糊升级 + 数字色，见下 3 帧。

画廊 B — 超屏内容（bg 缩放进程 + 滚动长列表）

cg01 没有可滚动的长地图（不像 cg03a 的路长卷）。飞行"升空"是 bg spine 随倍率 calcBgScale(m) 缩放（控制点 m1→1.0/2→1.125/10→1.5/100→1.875/1000→2.25，150ms lerp，pivot 右上 (234,−269)），足球 spine 钉死中心不动 —— 见下"bg 缩放进程"三档示意（非长卷、是拉近）。真正"超一屏"的是：AllPlayers 排行榜滚动(top50 × stride98 ≈ 4900px，VirtualScrollContainer) 与设置/注单滚动页(scroll 1344px+)。

③ 飞行场景机制（详见画廊 B + GDD §3）
  • 足球钉死中心：soccerSpine 固定 (390,330 局部)，panelScale=VIEW_H/660≈1.63；从不随倍率移动。
  • bg 缩放=升空错觉：bgSpine 按 calcBgScale(m) 缩放(pivot 右上 234,−269)，150ms lerp。**非相机/视差/长卷**。
  • 火球拖尾：fireball 每帧跟 soccer 的 bone15 worldX/Y(x−40/y+30)，3 级 scale+fps。速度线居中、模糊径向(球清晰外圈糊)。
  • 草屑/烟/尘：grass 挂 foot 骨持续喷(crash 爆)、smoke 挂 human 骨(crash)、dust 固定位(crash 后 300ms 放大)。
  • crash：足球 gameover 动画 + 中央定格倍率 + CRASHED 横幅；他人 cashout 飞抛物线彩带。
④ 控件渲染约定（资产 origin 见 ADD §6/§7）
  • ImageButton：四态 tint(default #CCCCCC 80% / hover #FFFFFF / press 0.95 缩放 + #B3B3B3 70% / disabled)。BetSwitcher 按下还 y 翻转。AutoCashout 开关开=绿 tint #80FF80。
  • 倍率/盈利数字走 SpriteDigits(5 色图集，字符 0-9/x/./,)，非字体；其余文字 runtime Pixi Text 叠(LanguageManager i18n)。
  • 程序化绘制(非图片)：TopBar 底色、toast/warning/language 渐变描边框(#65F4FA→#FEF988)、设置分割线、结算黑幕 vignette + 光条 tint、各面板 mask。
  • 语义色：BET/cashout 绿系 #4DD498；撞毁红 #FF6161；奖励金渐变 #FED027→#FFEF73(描边 #DDA23C)；数字 蓝(<2x)/绿(2-10x)/橙(≥10x)。
