③ 飞行场景机制（详见画廊 B + GDD §3）
  • 榴莲 Y 固定：Scene2Panel 局部 (380,330) scale 0.2，panelScale 1.794；**从不随倍率移动**。`cameraShaking.ts` 死代码、无相机/视口变换。
  • 三张滚动 bg：bg1/bg2/bg3 各 1080×2160 TilingSprite，tilePosition.y += 2px/帧 恒速、无视差；lv 切换 600ms 交叉淡化。
  • 氛围装饰(FlyingItems，Math.random 生成、装饰性)：lv1 气球×3 / lv2 热气球×2 / lv3 UFO×2，从屏顶外落到底销毁。
  • 周身特效(挂 spine 骨)：smoke(4×4) / 液体(10×2) / 情绪(4×4，lv 专属) / 升级 explode(5×2)+spark(4×2) / speedline(8×3 全屏)。
  • crash：UFO(5×3) 飞掠 + 光柱(色随 lv：橙/青/品红) + roundlight×4 + 榴莲切开。无碰撞检测，纯演出。
④ 控件渲染约定（资产 origin 见 ADD §5/§7）
  • ImageButton：四态 tint(normal#E5E5E5/hover#FFF/pressed#CCC/disabled#7F7F7F) + scale(按下 0.9/抬起 1.0)。部分按钮 ship hover 件但代码未用(btn_bet_hover/btn_cancel_hover 死件)。
  • 倍率/盈利数字走 SpriteDigits(5 色图集 1530×181，字符 0-9/x/./,)，非字体；其余文字 runtime Pixi Text 叠在 text-free 框上(LanguageManager.bindText i18n)。
  • 程序化绘制(非图片)：TopBar/History/PrizePool 底色块、toast/warning 渐变描边框、设置分割线、guide 卡(tilted-olive 橄榄绿斜卡)、各面板 mask —— 运行时 Graphics/Canvas 按 theme 画。
  • 语义色：BET 绿描边#22D37B / CANCEL 红#d03232 / FULL CASHOUT 金#C49029 / 50% CASHOUT 橙#D8755B / WAIT 灰#848484；数字 蓝(<2x)/绿(2-10x)/金(≥10x)。
