注释 — 屏覆盖 / HUD vs game content / 来源 / 缺口
  • 屏覆盖: ①Loading(宿主 React) ②主界面 HUD ③Game content(转轮/板/装饰) ④Free Spins 过场 ⑤Settings+帮助赔付+语言 ⑥弹窗 ⑦选注/账单 ⑧特效。均从代码合成,非截图。
  • HUD 与 game content 分离: Canvas_PlayerHUD_Vertical/Horizontal(HUD)与世界空间 GameBoard/Decorations+GameBoardUI_Placeholder(内容)在代码里是不同根,故分屏(②③)。
  • 来源: 主体 data.unity3d;设置/帮助/语言/账单=`ui_settingpage_pack`;弹窗=`ui_popuppage_pack`;免费过场=`gameplay_freegame_noticerpack`;特效=`gameplay_winreward_effectpack`;装饰/角色=`gameplay_decoration(_character_pack)`。Loading=宿主 React CSS。
  • 徽标: extracted=构建 RectTransform 合成 px; derived=世界空间(reels 屏幕y)/宿主 React 设计空间/布局组运行时。cr=有 CanvasRenderer(上屏可见);虚线=结构容器;半透=act=False(默认隐藏态)。
  • 缺口: 布局组(GridLayoutGroup)驱动的件(如 BetAmountSelectorPanel 4×3 注额钮)静态 px collapse 0×0,单件位=运行时,标 derived。角色 BaseGirl/FreeGirl=世界 Spine 无 px。
  • 占位: 余额/赢分/注额/乘数/次数/订单号=运行时下发,一律 <占位>。
